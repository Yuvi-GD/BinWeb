# Session Context - BinWeb Recovery

This file preserves the development context for BinWeb.

## Project Structure
- `tools/embedder.cpp`: Utility that converts the `index.html` static asset into a C++ hex byte array `web_content.h`.
- `core/NativeApp.cpp`: Wrapper class initializing WebView2 on Windows and navigating to a temporary local file to permit `localStorage` and HTML5 API operations.
- `core/NativeApp.h`: Interface for `NativeApp`.
- `platform/windows/main.cpp`: Entry point for Windows application.
- `web/index.html`: The HTML5/JS/CSS user interface.

## Current State
- Built successfully using MSVC / CMake under Release profile.
- Reverted JS logic to run pure `localStorage` and `IndexedDB` operations by serving the embedded HTML byte array through a temporary local file protocol (`file:///`).
- Handled origin restrictions successfully.
